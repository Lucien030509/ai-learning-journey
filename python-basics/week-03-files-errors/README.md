# Week 3: Files and Errors

You will create these files during lessons:

- `csv_missing_values.py`: read a small CSV file and identify missing values.
- `safe_file_reader.py`: handle a missing file without crashing.
- `log_helper.py`: write simple learning logs to a text file.

Key ideas: file paths, `with open(...)`, `try / except`, and reading error messages.
