# Python Basics

This folder is the foundation for COMP9001 and later data work.

- **week-01-control-flow**: variables, input, conditions, loops, and guessing games.
- **week-02-functions-collections**: functions, lists, dictionaries, and small data summaries.
- **week-03-files-errors**: reading files, handling bad input, and debugging.
- **week-04-oop**: classes and objects (a way to organise related data and actions).

We create each new exercise file during the lesson, after you understand its goal.
