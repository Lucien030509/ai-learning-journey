# AI Learning Journey

My learning repository for the University of Sydney Master of Artificial
Intelligence journey. It contains small programming exercises, course-aligned
data work, and progressively more complete AI projects.

## Current focus

- Python foundations: functions, lists, dictionaries, files, testing, and
  debugging.
- COMP9120-aligned SQL and relational database design.
- COMP5310-aligned data analysis and machine-learning foundations.
- Clear Git history and project documentation from the beginning.

## Learning plan

The active, version-controlled roadmap is in
[notes/ai-career-roadmap.md](notes/ai-career-roadmap.md). It is designed for
the path from beginner to an applied AI engineer with strong foundations and
frontier-AI literacy.

## Repository structure

```text
python-basics/   # Small Python exercises and weekly review programs
notes/           # Learning notes and the active roadmap
```

More folders are added only when a real course or project needs them.
